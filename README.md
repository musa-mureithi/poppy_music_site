# 🎵 Poppy Music Site

Poppy Music Site is a simple yet modern music website built with **HTML, CSS, JavaScript, and Python**.  
It allows users to browse, play, and enjoy music through a clean and responsive interface.

---

## 🚀 Features
- 🎧 Play and pause music with custom controls  
- 🔍 Search songs and artists (if implemented)  
- 📂 Organize tracks into playlists or categories  
- 📱 Responsive design (works on mobile & desktop)  
- 🎨 Stylish UI powered by CSS  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Django )  
- **Database:** SQLite (or any database you configured)
